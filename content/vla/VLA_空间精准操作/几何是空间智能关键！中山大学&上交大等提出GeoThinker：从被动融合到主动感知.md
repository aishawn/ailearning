# 几何是空间智能关键！中山大学&上交大等提出GeoThinker：从被动融合到主动感知，刷新多榜单纪录

在通往通用人工智能（AGI）的路上，空间智能一直被视为多模态大模型（MLLM）的“深水区”。虽然现在的模型能吟诗作画，但在面对复杂的 3D 场景时，往往还是会显得有些“空间感缺失”。为了补齐这块短板，研究者们开始尝试给模型喂入几何先验。不过，现有的方法大多是“一股脑儿”地把几何特征塞给模型，这种被动融合的方式不仅容易引入噪音，还常导致语义和结构的错位。

最近，来自中山大学、引望智能、上海交通大学等机构的研究团队提出了一种全新的框架——GeoThinker。该模型被命名为 “GeoThinker”，其中 “Geo” 代表几何（Geometry），“Thinker” 意为思考者，寓意模型不再是机械地接收数据，而是能够带着几何思维去主动理解世界。它将传统的“被动融合”范式转变为“主动感知”，让模型学会根据当前的推理需求，有选择性地去检索和利用空间线索。目前，该项目已在 GitHub 开源，并在多个空间智能榜单上刷新了纪录。

- **代码仓库**：[https://github.com/Li-Hao-yuan/GeoThinker](https://github.com/Li-Hao-yuan/GeoThinker)（已开源）
- **主要机构**：中山大学、上海交通大学、引望智能、MBZUAI

## 告别“被动”，拥抱“主动”

在讨论 GeoThinker 之前，我们先来看看目前的几何感知模型是怎么做的。

通常情况下，开发者会给大模型配备一个专门的 3D 编码器（比如 VGGT）。在处理视频或图像时，模型会将几何特征直接加在视觉特征上，或者通过简单的拼接让模型学习。这种方式就像是在嘈杂的菜市场里听人说话，虽然信息量大，但干扰项也多。

**范式对比**：如上图所示，作者对比了三种范式：

(a) 几何作为输入（Passive Fusion）：直接相加，容易产生语义-几何错位。  
(b) 几何作为监督（Passive Fusion）：通过对齐来学习，但推理时缺乏灵活性。  
(c) 主动感知（GeoThinker）：模型根据内部推理需求，自主辨别并检索任务相关的几何线索。

这种转变的核心逻辑在于：并不是所有的几何信息对当前任务都有用。比如，当你问“沙发在自行车的什么位置”时，模型应该关注这两个物体的边界和相对距离，而不是去分析地板的平整度。

## 核心技术：空间定位融合与重要性门控

GeoThinker 的架构设计非常精巧，它在不破坏大模型原生语义理解能力的前提下，植入了一个“空间大脑”。其核心在于如何让语义特征主动去“寻找”几何特征。

**架构概览**：

### 1. 空间定位融合（Spatial-Grounded Fusion, SGF）

这是 GeoThinker 的心脏。为了保证空间的一致性，研究团队设计了帧向约束（Frame-wise Constraints, FWC）的交叉注意力机制。

在 SGF 模块中，语义视觉特征作为 Query，而几何特征作为 Key 和 Value。为了防止跨帧特征的干扰，模型被强制要求在每一帧内部进行注意力计算。这意味着，当模型处理第一帧的图像时，它只能去查询第一帧对应的几何深度或结构信息。这种“帧严格”的设计确保了空间位置的精准对应，避免了视频处理中常见的特征漂移问题。

### 2. 重要性门控（Importance Gating, IG）

如果说 SGF 提供了查询的通道，那么**重要性门控（Importance Gating, IG）**就是那个聪明的“过滤器”。

并不是图像中的每一个像素都需要几何信息。GeoThinker 会根据当前的图像状态预测一个重要性分数，并将其转化为注意力偏差。在计算交叉注意力时，这个偏差会被加入到 Logits 中。通过这种方式，模型能够自动“高亮”物体边界、结构边缘等关键区域，而抑制掉像白墙、地板这类冗余的背景信息。最后，通过一个全局缩放（Global Scaling）系数，模型可以平滑地将这些几何线索注入到大模型的残差流中。

## 性能表现：刷新多项纪录

为了验证 GeoThinker 的实力，团队在多个严苛的基准测试上进行了实验，涵盖了从基础的空间感知到复杂的物理推理。

### 1. 空间推理的“天花板”

在 VSI-Bench（空间智能测试）上，GeoThinker 展现出了统治级的性能。在基础模式（Vanilla regime）下，GeoThinker 在 3B 和 7B 规模上均显著超越了基准模型 VG-LLM。

**VSI-Bench 结果**：而在扩展训练模式（Scaled regime）下，GeoThinker (Qwen3VL-8B) 直接跑出了 72.6 的惊人成绩，刷新了该榜单的纪录。

在更全面的 EASI 排行榜上，GeoThinker 位列全球第 6，在开源模型中名列前茅，甚至在某些维度上超越了参数量大得多的闭源模型。

### 2. 拒绝“投机取巧”：VSI-Debiased 测试

很多模型在测试时会通过语言先验（比如“猜”沙发通常靠墙）来回答问题，而不是真的看懂了图。为了剔除这种偏差，研究者使用了 VSI-Debiased 进行测试。

结果显示，GeoThinker 在 128 帧的长视频输入下依然稳健，拿到了 68.1 的分数。相比之下，即使是专门针对长视频优化的 Cambrian-S-7B 也只有 59.9 分。这证明了 GeoThinker 的空间理解是“真功夫”，而非依赖语言套路。

## 下游任务的“实战派”

GeoThinker 不仅在实验室榜单上表现优异，在具身智能和自动驾驶等实际应用中也展现了巨大的潜力。

- **具身智能（RoboRefer）**：在处理“把那个杯子放到桌子左边”这类指令时，机器人需要极强的空间定位能力。GeoThinker 帮助模型在 RefSpatial-Bench 上实现了 1.66% 的平均精度提升。
- **自动驾驶（NAVSIM）**：在复杂的驾驶场景中，准确判断障碍物的距离和位置至关重要。GeoThinker 显著增强了规划的安全性，在行人距离安全边际（PDMS）等关键指标上表现优异（从 81.6 提升至 83.6）。

## 可视化分析：它在看哪里？

为了直观理解模型是如何“主动思考”的，研究团队可视化了重要性门控生成的评分图。

**重要性分数可视化**：从热力图中可以清晰地看到，GeoThinker 自动忽略了平坦的地板、天花板和单调的墙面，而将注意力高度集中在物体的轮廓、边界以及物体间的连接处。这种行为与人类观察 3D 环境的方式高度相似——我们总是先寻找物体的边缘来构建空间感。

## 写在最后

GeoThinker 的成功给我们带来了一个重要的启示：在大模型时代，信息的“多”并不一定代表“强”，如何“聪明地取舍”才是关键。

通过将几何集成从“被动融合”转为“主动感知”，GeoThinker 不仅提高了计算效率，更重要的是，它让模型在处理 3D 世界时拥有了类似人类的“注意力机制”。这种“按需索取”的策略，或许正是下一代空间智能模型突破瓶颈的钥匙。

目前，GeoThinker 的代码和权重已经全部公开。如果你也对具身智能或 3D 视觉感兴趣，不妨去 GitHub 上 Clone 一下代码，亲自感受这个“带几何思维的思考者”究竟有多聪明。
