### CoCo: Code as CoT for Text-to-Image Preview and Rare Concept Generation

**arXiv**: 2603.08652 (2026年3月9日提交)

**核心问题**： 纯文本 Chain-of-Thought (CoT) 在文本到图像生成中规划粗糙，尤其对稀有概念（rare concept）、复杂布局、属性组合生成效果差，容易出现语义偏差。

**主要创新**： 提出 **Code-as-CoT (CoCo)**，将推理过程显式表示为**可执行代码**，作为中间结构化规划。

**方法关键点**：

- 三阶段流程：(1) 根据prompt生成可执行代码（指定场景布局、结构）；(2) 在沙箱环境中执行代码，渲染确定性草图（draft image）；(3) 以草图为基础进行精细图像编辑，输出高保真结果。
- 代码作为显式、可验证的中间表示，比纯文本更精确。

**亮点**： 在结构化基准（StructT2IBench）、单图生成（OneIG-Bench）、长文本基准上大幅超越直接生成（+41%~+68%），对稀有概念和复杂布局特别有效。代码开源。
