## 背景与动机：为什么要跳出自回归的“舒适区”？

过去几年，多模态智能的发展主要依赖于强大的预训练大语言模型（LLM）。为了让 LLM 听得见、看得懂，研究者们通常会给它挂载各种“感官”插件。虽然这种方式效果不错，但自回归架构本质上是单向的，既难以实现高效的并行化生成，也难以做到在生成过程中关注全局信息。

相比之下，**掩码离散扩散模型（Masked Discrete Diffusion Models, MDMs）**展现出了独特的魅力。与 AR 模型必须按顺序生成不同，MDM 支持并行解码，这意味着可以在一次前向传播中解码多个 Token，加速生成过程；此外，它的生成轨迹可以被灵活引导，从而实现更精准的语义控制。既然扩散模型在纯文本（如 Dream、LLaDA）和纯图像领域都玩得转，那能不能用它来构建一个“全模态”的统一大脑呢？

![Omni-Diffusion 处理各种双模态及多模态任务的示意图](https://pub-adba99cbc4cd4237a5ed7de21ad26f3c.r2.dev/md-img/d3ff8b5484a7e07b.png)

如上图所示，Omni-Diffusion 的目标就是打破模态间的壁垒。无论是语音转图像、口头视觉问答等多模态高级任务，还是传统的文本转语音（Text-to-Speech, TTS）、文本图像生成（Text-to-Image）等双模态基础任务，它都希望在一个统一的概率建模框架下完成。


## 方法详解：统一离散空间的“魔法”

Omni-Diffusion 的核心思想是将所有模态的数据都转化为离散的 Token，然后在这些 Token 组成的序列上进行掩码预测。

### 1. 模态的离散化（Tokenization）

要让一个模型处理多种模态，首先得让它们说“同一种语言”。

- **图像模态**：采用了 **MAGVIT-v2** 编码器，将图像压缩为 8192 种可能的离散 Token。
    
- **语音模态**：输入端使用 **SenseVoiceSmall** 提取语义特征，输出端则利用 **GLM-4-Voice** 的分词器将语音转化为 16384 种离散 Token。
    
- **文本模态**：继承自基础语言模型的词表。
    

### 2. 掩码离散扩散模型架构

模型的主干是拥有 70 亿参数的 **Dream-7B**。不同于自回归模型预测“下一个 Token”，Omni-Diffusion 通过掩码 Token 预测（Mask Token Prediction）学习多模态 Token 的联合分布。

### 3. 三阶段渐进式训练

为了让模型稳步掌握各项技能，团队设计了三步走的策略，逐步扩大模态范围：

- **第一阶段（视觉-语言预对齐）**：先让模型学会看图说话（Captioning）和文生图，把视觉和文本空间对齐。
    
- **第二阶段（多模态联合对齐）**：加入语音数据（ASR/TTS），让模型在视觉、语言、语音三者间建立联系。
    
- **第三阶段（能力强化）**：在团队自建的 **SDVI 数据集**上进行微调，专门强化“语音驱动的视觉交互”能力。
    

![三阶段训练流水线，展示了从单一对齐到全模态融合的过程](https://pub-adba99cbc4cd4237a5ed7de21ad26f3c.r2.dev/md-img/c51030ea530a5798.png)

特别值得一提的是，团队提出了**衰减尾部填充掩码（Attenuated Tail-Pad Masking）**策略。在处理变长生成时，通常会添加 Pad Token，但简单的掩码会导致模型过度关注 Pad。通过引入一个缩放因子  来降低 Pad 的掩码比例，模型能更专注于核心语义的生成。

### 4. 推理优化

扩散模型在推理时采用基于熵的解码策略。团队提出了几个非常实用的技巧：

- **位置惩罚（Position Penalty）**：扩散模型在并行解码时，往往会同时从序列首尾向中间生成。这在图像任务中会导致“镜像重复”现象（即图片顶部和底部出现雷同图案）。团队通过在推理初期对序列后部  个 Token 进行概率抑制，巧妙地打破了这种对称性，使画质更加自然。
    
- **特殊 Token 预填充**：在生成语音时，模型会在序列 25% 的位置预先埋入 [begin-of-speech] 标记。这相当于给模型画了条线：前 1/4 走文本逻辑，后 3/4 走语音逻辑。这种方式让模型在生成语音时能实时“参考”前方的文本语义，大幅提升了对话的连贯性。
    
- **自适应 Token 长度分配**：根据语音时长与文本长度的强相关性，动态调整 ASR 和 TTS 的初始掩码长度，既提升了性能又加速了采样。
    

## 实验与结果：不只是“全能”，更是“高性能”

研究团队在多个权威榜单上对 Omni-Diffusion 进行了全方位的“大考”。

### 语音任务：超越专门的语音模型

在语音识别和语音合成任务中，Omni-Diffusion 展现出了不错的实力。在 LibriSpeech 榜单上，其 WER（词错误率）显著优于同为 Any-to-Any 架构的 AnyGPT。

![语音任务性能对比表](https://pub-adba99cbc4cd4237a5ed7de21ad26f3c.r2.dev/md-img/3b14fa0bf2384625.png)

语音任务性能对比表

### 视觉理解与生成：全模态中的佼佼者

在视觉问答（Visual Question Answering, VQA）方面，Omni-Diffusion 在多个评测基准中拿到高分，不仅远超 AnyGPT，甚至与一些专门做理解的视觉 LLM 不相上下。在文生图任务中，其 CLIP Score 也达到了不错水平。

![视觉理解与生成任务的性能对比](https://pub-adba99cbc4cd4237a5ed7de21ad26f3c.r2.dev/md-img/4520b05b40dd16d9.png)

### 采样效率：10步就能画出好图

扩散模型最怕“慢”。但实验证明，Omni-Diffusion 在图像生成上非常有韧性。得益于并行解码，将采样步数从 256 步缩减到 **10步** 时，画质依然维持在较高水平。

## 写在最后

Omni-Diffusion 的出现，为多模态大模型的研究开辟了一条新路径。它有力地证明了：自回归并非通往全模态智能的唯一方案。通过巧妙地利用离散扩散模型的并行性和统一建模能力，我们同样可以构建出既能“读万卷书”又能“行万里路”的全能 AI。

目前，该项目已经在 GitHub 上开源，包含完整的训练流程和数据集说明。这种离散扩散模型的架构，未来是否会取代现有的自回归方案？抑或是两者会走向某种融合？这非常值得我们期待。如果你对高效推理和全模态交互感兴趣，Omni-Diffusion 绝对是一个值得深入研究的范本。
