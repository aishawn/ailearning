# 从“像素重建”转向“动作语义”：VLA-JEPA，让机器人从人类视频中参透“动作奥义”

*[图片]*

在具身智能领域，让机器人像人类一样通过观看视频来学习技能，一直是个极具吸引力的方向。然而，现有的视觉-语言-动作模型（Vision-Language-Action Model, VLA）在利用海量互联网视频进行预训练时，往往会陷入“像素陷阱”：它们太过于关注画面的光影变化或背景杂物，而忽略了动作背后的核心逻辑。为了解决这一难题，来自中国科学技术大学、中关村实验室、上海交通大学、清华大学等机构的研究团队联合提出了一种全新的预训练框架——VLA-JEPA。

该模型被命名为“VLA-JEPA”，其中 VLA 代表了其视觉-语言-动作的本质，而 JEPA 则代表了联合嵌入预测架构（Joint-Embedding Predictive Architecture, JEPA）。这一命名的核心意图在于，将原本纠结于像素重建的动作学习，转化为在潜在空间（Latent Space）中进行的预测任务。就像人类在观察他人操作时，大脑并不会去复刻每一个像素点的闪烁，而是“脑补”出物体状态的转换逻辑。

为什么现在的机器人预训练总是“学歪了”？
在深入了解 VLA-JEPA 之前，我们先聊聊当前 VLA 模型面临的尴尬处境。虽然我们可以给机器人看成千上万小时的人类操作视频，但这些视频往往带有严重的“噪声”：

像素偏差：模型往往在学习如何预测下一帧像素，结果它记住了光影、纹理和背景，却没记住手是怎么拿杯子的。
噪声运动：在第一视角视频中，相机的晃动往往比手部动作更剧烈，模型很容易把相机的位移误认为是动作。
信息泄漏：很多模型在训练时会“偷看未来”，它们直接把未来帧的信息塞进编码器，导致学到的“动作”成了未来画面的压缩包，而不是真正的操作逻辑。
流程复杂：为了修补上述问题，现有的方案往往需要极其复杂的多阶段训练，既难调优又容易崩溃。
作者通过可视化注意力图（Figure 6）敏锐地指出，像 LAPA 这样的传统方法，其潜在动作 Token 会关注极其密集的视觉信息，甚至包括与操作无关的桌面杂物，这就是典型的“像素纠缠”。

*[图片]*
作者认为，真正的“动作”应该是描述底层状态如何演化的变量。这就引出了 VLA-JEPA 的核心思想：无泄漏的状态预测（Leakage-free State Prediction）。

方法详解：在潜在空间中“脑补”未来
VLA-JEPA 的架构设计非常精巧，它不再强求模型去还原真实的画面，而是让它在“潜意识”里进行对齐。

*[图片]*
核心架构：VLM 与世界模型的合体
VLA-JEPA 采用了 Qwen3-VL 作为核心的大视觉语言模型主干，并结合了 SigLIP-2 视觉编码器。其独特之处在于引入了一组可学习的特殊 Token：（潜在动作）和 （具身动作）。

输入（Input）：当前时刻  的多视图观测图像  和语言指令 。
处理流程：VLM 将这些输入转化为潜在动作表示 。
输出（Output）：这些潜在表示被送入一个基于 Transformer 的“潜在世界模型”，用来预测未来的环境状态。
绝不“偷看”的 JEPA 风格对齐
这是 VLA-JEPA 最硬核的地方。在预训练阶段，研究者使用了一个冻结的 V-JEPA2 编码器作为“老师”，它负责生成未来帧的潜在表示  作为监督目标。

*[图片]*
关键点在于：未来的视频帧永远不会作为输入喂给 VLM 主干。VLM 只能看到当前这一帧，它必须通过“脑补”潜在动作，让世界模型预测出的未来状态  与“老师”生成的真实未来状态  尽可能接近。

从数学角度看，这实际上是在语义空间中最大化预测对数似然的证据下界（Evidence Lower Bound, ELBO）：

由于目标编码器  是确定的，KL 散度项消失，目标简化为在潜在空间中的重构损失。这种设计彻底杜绝了信息泄漏，强迫模型学习真正的动力学特征。

实验结果：不仅更强，而且更稳
研究团队在 LIBERO、SimplerEnv 以及真实世界的 Franka 机器人上进行了全方位的测试，涵盖了仿真到现实的各种挑战。

极强的鲁棒性
在更具挑战性的 LIBERO-Plus（专门测试鲁棒性的基准）中，VLA-JEPA 的优势更加明显。面对相机抖动、光照变化、背景干扰等 7 个维度的“折磨”，它依然保持了 79.5% 的平均成功率，而强大的  在这里仅有 53.6%。这有力地证明了，在潜在空间学习动力学，确实比在像素空间死磕要鲁棒得多。

真实世界的惊喜：学会了“失败重试”
在真实世界的 Franka 机器人实验中，研究者发现了一个非常有趣的现象。

*[图片]*
传统的模型如  或 ，如果第一次抓取失败，往往就“死机”或者继续执行后续动作。而 VLA-JEPA 竟然学会了“失败后重试”（repeated grasping）：如果没抓稳，它会自动张开夹爪再试一次。这种技能在机器人训练数据中很少见，但在人类视频中比比皆是。这说明 VLA-JEPA 确实从人类视频中汲取了有用的操作逻辑，并成功转化为了自己的动作策略。

*[图片]*
写在最后
VLA-JEPA 的成功给我们带来了一个重要的启示：在具身智能的学习中，“看”什么固然重要，但“怎么看”才是决定性的。通过舍弃像素级的繁枝末节，转而在潜在空间中追求状态转换的对齐，我们不仅能获得更强大的模型性能，还能让机器人学到那些难以用代码定义的“操作直觉”。

这种将 JEPA 思想引入 VLA 的尝试，无疑为利用互联网规模的非结构化视频数据开辟了新的道路。未来，当机器人能够真正读懂人类视频中的每一个细微动作逻辑时，通用机器人的时代或许就不远了。
