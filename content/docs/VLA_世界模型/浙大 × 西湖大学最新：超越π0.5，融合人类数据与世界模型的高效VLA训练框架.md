# FRAPPE：世界建模注入通用策略的多未来表征对齐

浙大、西湖大学、港科大（广州）、华南理工、上科大、清华等提出 **FRAPPE**（Future Representation Alignment via Parallel Progressive Expansion），将世界建模能力注入 VLA 策略，在 RoboTwin 与真机任务上超越 RDT-1B、π0.5，并在长时程与未见场景中表现更强泛化。

- **论文**：https://arxiv.org/abs/2602.17259  
- **主页**：https://h-zhao1997.github.io/frappe/

---

## 问题与思路

- **问题**：现有方法（1）训练目标过度强调像素级重建，限制语义与泛化；（2）推理依赖预测的未来观测，易误差累积。
- **方案**：通过**并行渐进扩展**实现多未来表征对齐——不做显式像素级未来生成，而是对**未来观测的隐空间表征**做对齐，使动作与视觉在统一语义空间中理解；同时用**并行扩展**（多输入流、多教师表征）避免单一归纳偏置。

---

## 方法要点

- **训练机制**：隐式世界建模，对齐未来观测的隐空间表征。
- **模型结构**：将单流 VLA 扩展为多流并行前向，每流对齐一独立视觉编码器（实验采用 3 路：DINOv2、CLIP、ViT），多教师表征联合学习。
- **训练阶段**：Mid-Training 全参数微调、与 Theia 对齐；Post-Training 引入并行扩展（多 prefix + LoRA），转为混合专家并做多教师特征对齐。

---

## 实验要点

- **仿真**：RoboTwin 8 子任务平均成功率超越 naive fine-tuning 的 RDT-1B、π0、π0.5 及 DP、VPP；RDT-130M 经 FRAPPE 后与 RDT-1B 持平，小模型也受益。
- **真机**：四类双臂操作任务上超过 RDT-1B 与 π0.5，未见复杂场景泛化更好；长程三子任务中 FRAPPE 与 π0.5 相当，RDT-1B 无法按序完成。
- **人类数据**：Ego (Web) 与 Ego (Task) 人类视频均可提升下游成功率，降低遥操作数据需求。
- **效率**：并行扩展带来额外参数与计算，推理延迟仍与 RDT-1B 接近，显存可接受。

FRAPPE 通过表征对齐与多源数据融合，在可接受推理开销下提升泛化、长时序与隐式世界建模能力。
