# LingBot-VA：蚂蚁灵波自回归视频-动作世界模型，超越 π0.5 约 20%

蚂蚁灵波开源 **LingBot-VA**，自回归扩散**视频-动作**世界模型，将大规模视频生成与机器人控制深度融合：在预测未来状态的同时直接推演动作序列，潜空间想象与动作推理协同，支持「一边推演、一边行动」。相比 π0.5 等顶流操作模型，准确率平均提升约 **20%**，高难度任务约 **30–50 条**数据即可适配。

## 架构要点

- **Video-Action**：不是 Vision-Action，而是世界模型方案；自回归扩散，视觉动态预测与动作推理统一到单一交错序列。
- **三部分**：自回归视频-动作世界建模（潜空间动态 + 从预测状态解码动作）；LingBot-VA 统一架构与训练；实时部署与异步推理。
- **流程**：自回归视频生成根据当前观测与语言指令预测未来帧 → IDM 从预测视频解码动作 → 执行后用真实观测替换视频 KV-cache，实现闭环。
- **自回归优势**：因果注意力与 KV-cache 维持长时程记忆；单向依赖与闭环一致；chunk-wise 并行 + chunk 间自回归，兼顾实时与纠错。
- **双流 DiT + MoT**：视频流与动作流双 Transformer，基于条件 flow matching 的自回归预测；MoT 实现视频与动作深度融合又保持参数独立。动作流更浅，因动作分布更简单。
- **异步推理**：用前向动力学模型（FDM）「想象」执行当前动作后的观测，据此预测下一动作，再用真实观测对齐「想象」观测，在提速的同时保持闭环稳健。

## 实验要点

- **真机**：制作早餐、拆快递、插入细管、拾取螺丝、叠衣服、叠裤子等；长程记忆与任务连贯性优于 π0.5，拆快递成功率提升约 40%，拾取螺丝约 20%，叠裤子等柔性任务显著提升。
- **RoboTwin 2.0**：Easy 92.9%、Hard 91.6%，超越 π0、π0.5、X-VLA、Motus 等；长时程（3 级）简单/困难场景分别提升约 8.2%、9.1%。
- **LIBERO**：LIBERO-Long 成功率 **98.5%**，当前领先。
- **数据效率**：10 条数据即可见效，「制作早餐」进度比 π0.5 高约 15.6%，RoboTwin 2.0 高约 10.3%；得益于视频-动作联合建模与物理先验。
- **记忆与泛化**：擦拭盘子六次、搜索盒子等记忆任务明显优于 π0.5；新物体与 OOD 空间泛化更强，世界模型通过视频预测学习可迁移的物理先验。

## 小结

LingBot-VA 将世界模型直接用于机器人操作并开源，与 LingBot-World、LingBot-VLA、LingBot-Depth 等共同构成蚂蚁灵波在「世界模型 + 具身操作」方向上的能力矩阵。
